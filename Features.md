Based on the PRD, here's a comprehensive list of features to be built for the Ariya web app, categorized by user type:
1. User-Facing Features (Event Planner / Guest)
1.1. User Management & Onboarding:
User Registration and Login
Social Login Integration (Google, Facebook)
Password Reset and Account Recovery
User Profile Management (Name, Contact Information, Profile Picture)
Dashboard (Overview of events, upcoming tasks, notifications)
1.2. Event Planning Tools:
Event Creation and Dashboard (Event Name, Type, Date, Location)
Guest List Management:
Import/Export Guest Lists (CSV)
Add/Edit/Delete Guests
RSVP Tracking
Guest Segmentation (e.g., Family, Friends, VIPs)
Dietary Restrictions and other notes per guest
Budget Tracker:
Categorize expenses (e.g., Venue, Catering, Entertainment)
Set budget limits per category
Track actual vs. budgeted expenses
Payment tracking and reminders
Seating Arrangement Tool:
Drag-and-drop interface for table/chair placement
Assign guests to tables/seats
Printable seating charts
Event Checklist/Timeline Generator (pre-populated templates for different event types)
Vendor Management within a Specific Event (Assigning booked vendors, tracking payments)
Task Management (Create custom tasks, assign due dates, track completion)
1.3. Vendor & Venue Discovery and Interaction:
Advanced Search & Filtering for Vendors/Venues (Keyword, Category, Location, Price Range, Ratings, Availability, Specific Features)
View Detailed Vendor Profiles (Business Name, Description, Contact Info, Service Categories, Portfolio, Pricing, Availability Calendar, FAQs, Reviews and Ratings)
Vendor Inquiry and Quote Request System
In-platform Messaging System for Vendor-Client Communication
Vendor Booking and Confirmation Workflow
Vendor Rating and Review System (post-event)
Receive notifications for important event milestones or vendor responses
1.4. Personalization & Sharing:
Customizable Event Website Generator:
Theme selection and customization (colors, fonts)
Ability to add event details (date, time, location, agenda)
Photo gallery
"About Us/Couple" section
RSVP form integration
Link to Event Registry
Shareable URL
Event Registry Creation:
Link to external gift registries (e.g., Amazon, Zola)
Option for cash fund contributions
Thank-you note tracking
1.5. Event Planner AI (Intelligent Planning Assistance):
Event Idea Generation:
AI to take user input (e.g., number of guests, desired atmosphere, occasion) and suggest suitable event types or themes
Provide inspirational content and brief descriptions for suggested event types
Budget Estimation & Allocation:
AI to provide a high-level budget estimate based on event type, location, and guest count
Ability for users to input a fixed overall budget
AI to intelligently allocate the user's budget across predefined vendor categories
Provide a visual breakdown (e.g., pie chart) of the suggested budget allocation
Allow users to manually adjust the AI's budget allocation, with the AI recalculating other categories dynamically
Budget-Based Vendor Recommendations:
Integrate AI with the vendor database to filter and recommend vendors whose pricing aligns with the user's allocated budget for specific categories
Present a list of recommended vendors, clearly indicating if they fall within, slightly above, or significantly above the allocated budget
Ability to "lock in" AI-recommended vendors, which would then automatically update the budget tracker
For vendors outside the budget, AI should suggest similar, more affordable alternatives or provide options for adjusting the budget
Cost Optimization Advice:
AI to provide actionable tips and advice on how to save costs within different event aspects
Proactive suggestions to optimize budget based on user's current planning stage
Feedback Loop for AI Improvement:
Allow users to provide feedback on the accuracy and helpfulness of AI recommendations
1.6. Guest-Specific Features:
Access an event's website to view details, RSVP, and get directions
Access an event's registry to contribute or purchase gifts
2. Event Vendor-Facing Features
2.1. User Management & Onboarding:
User Registration and Login
Social Login Integration (Google, Facebook)
Password Reset and Account Recovery
User Profile Management (Name, Contact Information, Profile Picture)
Dashboard (Overview of inquiries, bookings, profile performance)
2.2. Vendor Profile & Service Management:
Create a detailed business profile highlighting services, portfolio, and pricing
List services under clearly defined categories (e.g., Venues, DJs & Bands, Catering Services)


Specific categories include: Venues, DJs & Bands, MCs & Hosts, Catering Services, Decorators, Florists, Car Hire, Photographers, Videographers, Event Planners, Equipment Rentals, Party Rentals, Hair & Makeup Artists, Wedding Dress & Attire, Stationery & Invitations, Security Services, Clean-up Crew, Travel & Accommodation
Manage availability calendar to prevent double-bookings
Update pricing and service offerings
Receive reviews and ratings from clients
2.3. Inquiry & Booking Management:
Receive booking inquiries and quote requests from event planners
Respond to inquiries and send quotes through the platform
Manage bookings and view upcoming events
In-platform Messaging System for Vendor-Client Communication
2.4. Payment & Monetization:
Receive payments securely through the platform
Upgrade to a premium listing to increase visibility
3. Admin-Facing Features (Implied for Platform Management)
While not explicitly detailed as a separate user type, the PRD mentions:
Vendor/Venue Listing Admin Approval Process
Reporting and Analytics for Vendor Subscription Performance
Management of Transaction Fee Application
Management of Vendor Subscription Tiers
